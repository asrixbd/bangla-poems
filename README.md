# নিভৃত আলাপ

শহর ঠাসা হাজার চেনা মুখে,
সবাই ছোটে নিজের মতো করে।
তবুও এক পাথর চাপা বুকে,
অচিন পাখি একলা ডানা ঝরে।

জানলার ঐ এক চিলতে রোদ,
বিকেল হলে অভিমানে মরে।
পুরানো সেই ভুলের প্রতিশোধ,
স্মৃতি হয়ে দরজাতে কড়া নাড়ে।

চায়ের কাপে ধোঁয়া ওঠা জল,
শীতল হয় একলা থাকার ঘোরে।
চোখের কোণে জমলে মেঘের দল,
বৃষ্টি নামে মনের গহীন চরে।

ঘরের কোণে আধো-অন্ধকার,
দেয়াল জুড়ে নিজের ছায়া ভাসে।
কেউ শোনে না কান্নার চিৎকার,
শুন্য ঘরও বিদ্রূপে আজ হাসে।

ডায়েরিটার হলুদ হওয়া পাতা,
জমে আছে অনেক কথা তাতে।
ব্যথার রঙে একাকীত্ব গাঁথা,
জেগে থাকে গভীর নিঝুম রাতে।

ফোনের পর্দায় নামগুলো সব স্থির,
কেউ তো আর ডানে না তোমায়।
এই বুঝি সেই জনমানুষের ভিড়,
যেখানে মন হারায় অবেলায়।

বালিশ ভেজা দীর্ঘশ্বাসের সুর,
নিঃশব্দে বালুচর গড়ে।
স্বপ্নগুলো যাচ্ছে অনেক দূর,
বালির ঘর যেমন ভেঙে পড়ে।

তবুও মানুষ একাই বাঁচতে শেখে,
নিজের সাথে সন্ধি করে শেষে।
একাকীত্বকে আপন করে রেখে,
হাসতে হয় ঐ মিথ্যে হাসির বেশে।
